VFPE Manual

The Visual Functional Program Editor (VFPE) is a tool for constructing and editing statically typed, purely functional programs in a diagrammatic and interactive environment.

Within the VFPE, functional expressions (and entire programs) are represented as syntax trees. The nodes of the tree represent various sorts of subexpressions such as variables, function applications, variable abstractions, built-in primitive expressions and undefined "placeholder" values. Programs are constructed and edited through a "drag-and-drop" interface.

Type information is maintained for each subexpression; it is available for inspection by the programmer and is used to verify the type correctness of the program during construction.

A simple interpreter allows single-step execution of expressions while in source form. Program construction can be freely intermixed with expression evaluation.

Expressions can be saved and loaded to and from files, and programs can be output as Haskell source code.

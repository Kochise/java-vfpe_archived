"How Do I ..."

This page describes how to perform a number of common programming operations.

How do I load an expression ?

  1. Click the "load" button on the pallet.
  2. Drop the expression where needed (onto the placeholder of the prelude node for an entire program).

How do I save an expression ?

  1. Open the control panel of the root node of the expression to be saved. Click the "copy" button to grab a copy of the expression.
  2. Drop the expression into the "save" control on the pallet. Note that only expressions with no free variables can be saved.

How do I write a Haskell code fragment ?

  1. Open the control panel of the root node of the "let" expression to be saved. Click the "copy" button to grab a copy of the expression.
  2. Drop the expression into the "write Haskell" control on the pallet. The bindings of the let expression become top-level Haskell definitions in a text file.

How do introduce a local definition at some point ?

  1. Grab a let expression from the pallet.
  2. Drop the expression onto the root node of the expression where the definitions are needed.

How do I introduce a constructor pattern (eg "(x : xs)" for non-empty list) ?

  1. Grab an instance of the constructor function from the pallet.
  2. Drop the expression onto the variable pattern being updated (variables can be renamed from the default values using the binding's control panel).

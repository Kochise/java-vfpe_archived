Visual Syntax (under construction)


see chapter 3 of thesis
Individual nodes, representing parts of the functional program's abstract syntax tree, fall into various catagories. Broadly speaking, these are:
  • Values. Appear as rectangular syntax nodes. Can be grabbed and dropped. All have cut, copy and evaluation operations.
  • Bindings. Appear as syntax nodes with rounded ends. Are always associated with some abstration value node. All have rename operations, and can be clicked on to created a named variable.
